## Run the server

1.  Create a `.env` file in the root folder
2.  See `.env.example` and replicate the values on `.env`
3.  `npm run start`

## Run the development server

1.  Create a `.env` file in the root folder
2.  See `.env.example` and replicate the values on `.env`
3.  `npm install`
4.  `npm run dev`
